## 文本格式

### 基本格式

MCPHub 文档支持标准的 Markdown 文本格式：

```md
**粗体文本** - 用于强调重要概念
_斜体文本_ - 用于强调或引用
`行内代码` - 用于命令、配置键或代码片段
~~删除线~~ - 用于标记过时的内容
```

**粗体文本** - 用于强调重要概念  
_斜体文本_ - 用于强调或引用  
`行内代码` - 用于命令、配置键或代码片段  
~~删除线~~ - 用于标记过时的内容

## 最佳实践

1. **标题层级**：保持清晰的标题层级结构
2. **代码示例**：为所有代码块指定语言
3. **链接检查**：确保所有内部链接有效
4. **图片描述**：为图片添加有意义的 alt 文本
5. **一致性**：在整个文档中保持术语和格式一致
